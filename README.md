# Task 4 - Windows Firewall Configuration

## Objective
Configured Windows Defender Firewall to block inbound traffic on Telnet port (23).

## Tool Used
- Windows Defender Firewall with Advanced Security

## Steps Performed
1. Opened Windows Defender Firewall.
2. Viewed existing inbound rules.
3. Created a new inbound rule to block TCP port 23.
4. Verified the rule was enabled.
5. Tested the configuration.
6. Removed the test rule.

## Outcome
Successfully learned how to create, verify, and remove firewall rules using Windows Defender Firewall.
